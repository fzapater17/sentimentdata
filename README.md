# Reddit Sentiment Analysis 
This project analyzes Reddit discussions to identify fan sentiment trends and their potential impact on match attendance. By scraping and analyzing comments, the goal is to uncover key concerns that affect ticket purchases, season ticket retention, and overall fan engagement.

# Repository Structure and Content

## redditSentimentAPIRem.ipynb
Extract Reddit posts and comments and perform sentiment analysis using OpenAI API. 

## edaSentiment.ipynb
Compare Reddit sentiment trends with match attendance and team performance.

## unstructuredAnalysis.ipynb
Analyze the Reddit posts and comments by applying unstructed data analysis techniques.
